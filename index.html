settings.json 配置文件

Windows: %USERPROFILE%\.claude\settings.json
Mac/Linux: ~/.claude/settings.json
settings.json 
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "your-api-key-here",
    "ANTHROPIC_BASE_URL": "https://api.kkyyxx.xyz"
  }
}

3

4
使用：运行 claude
